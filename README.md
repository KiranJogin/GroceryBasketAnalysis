# 🛒 Grocery Purchase Basket Analysis

An AI-powered retail analytics system that analyzes customer purchase behavior, generates product recommendations, and provides business insights using market basket analysis.

---

## 📌 Project Overview

This project focuses on analyzing grocery transaction data to identify patterns in customer purchases. It uses data analytics and association rule concepts (support & confidence) to recommend products and improve retail decision-making.

---

## 🎯 Objectives

- Analyze customer buying patterns from transaction data  
- Identify frequently purchased product combinations  
- Build a recommendation system for cross-selling  
- Implement a smart checkout with discount system  
- Visualize insights through an interactive dashboard  

---

## ⚙️ Features

- 🔐 User Authentication (Login/Register)
- 🧺 Smart Checkout System
- 🎯 Product Recommendation Engine
- 📊 Market Basket Analysis (Support & Confidence)
- 💰 Coupon & Discount Engine
- 🧾 Receipt Generation System
- 📈 Sales & Profit Dashboard
- 👥 Customer Insights
- 📁 Downloadable Reports
- 🗄️ Database Admin Panel

---


## 🛠️ Tech Stack

- **Language:** Python  
- **Frontend/UI:** Streamlit  
- **Database:** SQLite  
- **Libraries:**  
  - Pandas  
  - NumPy  
  - Plotly  
- **Concepts:**  
  - Market Basket Analysis  
  - Association Rules (Support, Confidence)  
  - Data Visualization  

---

## 📊 How It Works

1. Data is generated or loaded into the system  
2. Stored in SQLite database  
3. Processed using Pandas  
4. Market Basket Analysis identifies item relationships  
5. Recommendation engine suggests products  
6. Results are visualized in dashboard
